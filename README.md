# 作曲酒場（仮） / Composition Tavern

ランダムに配布される **コード進行カード** を組み合わせて楽曲を作る、ボードゲーム風の作曲アプリです。
プレイヤーは作曲家となり、客から依頼された楽曲を制作します。制作中には「店長カード」による
無茶振りや制約が発生し、限られたカードの中で曲を完成させます。

> 正解を作るのが目的ではなく、**発想の拡張・即興作曲の支援・セッションのお題作成・仲間との創作体験** を楽しむためのツールです。

## 遊び方

1. **新しい依頼** ボタンでゲームを開始（客・お題・曲構成・手札・店長カードが自動生成）。
2. **客カード**（依頼）と **お題カード**（ジャンル / テンポ / ムード / シチュ）を確認。
3. **店長の無茶振り**（構成変更・コード制約・転調・ジャンル変更）が発動。
4. **手札** のコード進行カードを **曲構成ボード** のスロットへドラッグ＆ドロップ。
   - スロットのセクション（A/B/サビ/C）と一致するカードのみ配置可能。
   - 配置済みカードはクリックで手札に戻せます。
5. **▶ コード進行を再生** で完成した曲を Tone.js で再生（お題のテンポで演奏）。

## 技術スタック

- **Next.js 16** (App Router) + **TypeScript**
- **Tailwind CSS v4**（酒場・木製テーブル風テーマ）
- **Zustand**（状態管理）
- **Tone.js**（コード進行再生）
- **dnd-kit**（ドラッグ＆ドロップ）

データはすべてローカル JSON（`src/data/`）。ログイン・DB不要。

## ディレクトリ構成

```
src/
├─ app/                 # ページ・レイアウト・グローバルCSS
├─ components/          # 各カード / ボード / 手札 / コントロール UI
├─ hooks/               # usePlayback（再生制御）
├─ store/               # gameStore（Zustand）
├─ lib/
│  ├─ music.ts          # ローマ数字 → コード名 / 音名 変換、転調
│  ├─ audio.ts          # Tone.js 再生エンジン
│  ├─ game.ts           # セッション生成・店長カード適用
│  └─ random.ts         # 抽選ユーティリティ
├─ data/
│  ├─ customers.json    # 客カード
│  ├─ themes.json       # お題カード
│  ├─ chords.json       # コードカード（A20 / B15 / サビ20 / C10）
│  └─ specials.json     # 店長カード
└─ types/               # 型定義
```

## 開発

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # 本番ビルド
npm run lint     # ESLint
```

## コード進行の仕組み

カードのコードは **ローマ数字**（`I` `V` `vi` `IV` など）で保持し、セッションごとに決まる
キー（例: Key=C → `C G Am F`）へ変換して表示・再生します。`7th` / `sus4` / 借用和音（`bVII`）/
オンコード（`IV/V`）などの表記にも対応。店長カードの転調（半音上 / 全音上 / 平行調）はラストサビ以降に適用されます。
