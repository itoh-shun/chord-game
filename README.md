# 🎛️ CHORD STUDIO

コード進行を**組んで・アレンジして・バンド即興セッション**にも使える、ポップな作曲サポートアプリ。
スマホ前提のゲーミングDTM。

📱 **https://itoh-shun.github.io/chord-game/**

## 2つのモード

### 作曲モード
- セクション（Aメロ/Bメロ/サビ…＝展開）を並べて曲を構成
- コードをタップして **度数・種類（メジャー/マイナー/7th/maj7/m7/sus4/6/add9/dim/aug）・借用(♭♯)・長さ(拍)** を編集
- 「定番を流し込む」で王道進行・カノン進行・小室進行などを一発入力
- キー / テンポ / 楽器 を変更して再生（再生位置をハイライト）

### ジャムモード
- 進行をループ再生。**今のコード**を大きく、**次のコード**も表示
- **キー即転調（±）** でその場でリハ移調
- **ソロの目安スケール**表示、**カウントイン**でバンド開始を合わせやすい

## 共通機能
- 楽器はサンプリング音源（ピアノ/アコギ/エレキ）＋シンセ（Tone.js）
- 簡単なドラム（キック/スネア/ハイハット）
- ギターのコードダイアグラム表示
- **MIDI書き出し**（コード＋ベースの2トラック）

## 技術
- Next.js 16 (静的書き出し) / TypeScript / Tailwind v4 / Zustand
- Tone.js（再生）・@tonejs/midi（書き出し）・@tombatossals/chords-db（運指）
- ローカル完結（ログイン/DB不要）、GitHub Pages で配信

## 開発
```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # 静的書き出し (out/)
npm run lint
```

`main` への push で GitHub Actions が自動ビルド＆Pagesへデプロイします。
