# ✦ BLOOM — 触れて咲く音

画面の**どこを触っても光の粒が咲き、心地よい音**が鳴る。
触れた点は**生成的にループ**してリピートし、光のリング・粒子・星座のような線が
音と完全同期して育っていく——ボタンではなく**画面全体が楽器**の、
音×光インタラクティブ・プレイグラウンド。

📱 **https://itoh-shun.github.io/chord-game/**

## あそびかた
- **どこでもタップ／なぞる** → 光の粒が生まれて発音。上ほど高音、左右でステレオ。
- 生まれた粒は**一定周期で鳴り続け**、重ねるほど曲が生成的に育つ。
- 音は常にスケール上から選ばれるので**外れない／いつも心地よい**。
- **ムード切替（夜 / 陽 / 夢 / 和）** で音階と色彩が一変。
- **🌧 自動** で手を離しても勝手に咲き続ける（作業・チル用）。**✕ 消す**でリセット。

## 仕掛け
- 触れた点＝ノード。誕生からの位相でループ発音し、近い点どうしは線でつながる（星座）。
- 発音の瞬間に粒が強く光り、リングが広がる。背景はムードの色相がゆっくり呼吸。
- スケールは五音音階や陰旋法など（夜=マイナーペンタ / 陽=メジャーペンタ / 夢=サス系 / 和=陰旋）。

## 技術
- Next.js 16（静的書き出し）/ TypeScript / Tailwind v4
- Canvas 2D（加算合成のグロー描画・requestAnimationFrame）
- Tone.js（PolySynth＋FMベル → ディレイ → リバーブ）
- ローカル完結・ログイン不要、GitHub Pages 配信

## 開発
```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # 静的書き出し (out/)
npm run lint
```
`main` への push で GitHub Actions が自動ビルド＆Pages へデプロイ。
