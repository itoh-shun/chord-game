# 🎛️ CHORD STUDIO

コード進行を**組んで・アレンジして・バンド即興セッション**にも使える、ポップな作曲サポートアプリ。
スマホ前提のゲーミングDTM。

📱 **https://itoh-shun.github.io/chord-game/**

## モード

### つなぐ（カードを組み合わせて作曲）🃏
- **Aメロ × Bメロ × サビ** の進行札を ◀▶ でめくって組み合わせ → 即・曲に
- 「**つなげて作曲 9,600曲**」の組み合わせカウンター（キー違いを入れればさらに増える）
- 🎲 全部シャッフル / ✏️ 編集へ（作曲モードで手直し）
- 各札はギターコードダイアグラム付き

### あそぶ（お題チャレンジ）🎮
- お客さんの「お題（ジャンル/ムード/ルール）」に合う進行を作って**提出**
- 音楽理論＋お題マッチで **★1〜3＋点数＋コメント**評価、**🪙コイン**獲得
- コインが貯まると**楽器がアンロック**（進捗はブラウザに保存）

## ツールの2モード

### 作曲（コード進行のアイデア創出）
- **💡 アイデア生成**：ムード（明るい/切ない/エモい/おしゃれ/激しい）を選んで、音楽理論ベースの進行を**自動生成**（何度でもリロール）
- 各展開ごとに **💡生成 / ↔代理コード / ✨おしゃれ化（7th）** でアレンジ
- コード編集時に **「流れに合う候補」** を提案（タップで置換）
- セクション（展開）を並べて曲を構成

### ジャムモード
- 進行をループ再生。**今のコード**を大きく、**次のコード**も表示
- **キー即転調（±）** でその場でリハ移調
- **ソロの目安スケール**表示、**カウントイン**でバンド開始を合わせやすい

## 共通機能
- 楽器はサンプリング音源（ピアノ/アコギ/エレキ）＋シンセ（Tone.js）
- 簡単なドラム（キック/スネア/ハイハット）
- ギターのコードダイアグラム表示
- **MIDI書き出し**（コード＋ベースの2トラック）

## 技術
- Next.js 16 (静的書き出し) / TypeScript / Tailwind v4 / Zustand
- Tone.js（再生）・@tonejs/midi（書き出し）・@tombatossals/chords-db（運指）
- ローカル完結（ログイン/DB不要）、GitHub Pages で配信

## 開発
```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # 静的書き出し (out/)
npm run lint
```

`main` への push で GitHub Actions が自動ビルド＆Pagesへデプロイします。
